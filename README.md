# Word Count

## How to run

1. Get to the right directory:

   ```bash
   cd frontend
   ```

1. First, install dependencies:

   ```bash
   npm install
   ```

1. Then either run with npm:

   ```bash
   npm start
   ```

1. or run with `docker-compose`:

   ```bash
   docker-compose up --build
   ```

1. Finally check it out on `localhost:3000`
